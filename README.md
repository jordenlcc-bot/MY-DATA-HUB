# MY DATA HUB · 全马人民 AI 数据村

> 「低门槛，马上能帮到大家也用得上 AI。」

MY DATA HUB 想做的，是一个给普通人用的 AI 数据村：  
不需要会写代码，不需要懂 Git，  
只要会登录、上传简单资料，按几个按钮，  
就可以让 AI 帮你写文案、整理名单、做简单分析。

## 为什么不是 GitHub，而是 MY DATA HUB？

GitHub 是给会写代码的工程师用的。  
MY DATA HUB 是给不会写代码的普通人用的。

在 GitHub，你需要懂：

- Git、branch、commit、pull request  
- 看 code、改 code

在 MY DATA HUB，你只需要：

- 登录账号  
- 上传简单的表格或资料  
- 按几个按钮，让 AI 帮你做事

一句话对比：

- GitHub = 代码宇宙，工程师的家  
- MY DATA HUB = 数据 + AI 的村子，uncle、小生意、老师、学生都能进来
