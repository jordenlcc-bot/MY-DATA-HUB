# MY DATA HUB · Malaysia AI 数据库

MY DATA HUB 是一个给马来西亚普通用户使用的 AI 数据库平台。  
不是只给工程师或程序员，而是给平常少用电脑、少用手机 app 的 uncle、auntie、小生意老板也能上手。

这里的重点很简单：

- 把复杂的 AI 和数据，整理到一个很简单的网页里面  
- 用户只需要会登录、点按钮、填少少资料  
- 就可以让 AI 帮忙做一些实际有用的事情（例如写宣传文案、整理客户名单、看生意大概情况）

市面上很多 AI / 数据工具都做得很专业，一般人很难用。  
MY DATA HUB 想做的是一个**容易理解、容易开始**的平台，  
让不熟技术的人，也可以一步一步跟上 AI 的发展，而不会被完全丢在后面。
